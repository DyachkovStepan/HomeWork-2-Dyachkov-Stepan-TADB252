# Менеджер рецептов и списков покупок

Проект представляет собой консольное приложение на Python для управления рецептами, подсчета количества ингредиентов в зависимости от порций и автоматического формирования списка покупок. В проект входят проверка приложения при помощи unit-тестов

# Использование
Для проверки работы кода и запуска тестов выполните в терминале команду:
```bash
py -m pytest test_recipes.py -v
```
